این کد شامل دو بخش اصلی است: یک بخش پایتون که با استفاده از فریم‌ورک Flask پیاده‌سازی شده و یک وب‌سرویس برای آپلود و پردازش تصاویر فراهم می‌کند و یک بخش PHP که یک فرم آپلود فایل در مرورگر فراهم می‌کند و با استفاده از cURL درخواست آپلود فایل را به وب‌سرویس پایتون ارسال می‌کند.

JSON تنها برای ارسال پاسخ از سرور پایتون به PHP استفاده می‌شود.

محتویات پروژه شامل فایل‌های زیر است:

app.py - فایل اصلی پایتون که شامل کدهای مربوط به Flask می‌باشد.
upload.php - فایل PHP برای ایجاد فرم آپلود و ارسال درخواست به سرور Flask.
Pipfile و Pipfile.lock - فایل‌های مربوط به مدیریت وابستگی‌های پروژه پایتون.
تصاویر و فایل‌های مرتبط با آپلود - شامل فایل‌های تصویری آپلود شده و فایل‌های موقت.


ساختار و داکیومنت پروژه


1. راه‌اندازی محیط توسعه
برای راه‌اندازی محیط توسعه، باید پیش‌نیازهای پروژه را نصب کنید. در این پروژه از Pipenv برای مدیریت وابستگی‌ها استفاده شده است.

نصب Pipenv:

pip install pipenv


نصب وابستگی‌ها:

install xampp and on mysql and apache

[packages]
hezar = {extras = ["all"], version = "*"}
torch = "==2.0.0"
flask = "*"
jsonify = "*"

[dev-packages]

[requires]
python_version = "3.11"


2. فایل app.py

3. فایل upload.php

4. اجرای پروژه
برای اجرای این پروژه، مراحل زیر را دنبال کنید:

راه‌اندازی سرور Flask:

به دایرکتوری پروژه بروید و محیط Pipenv را فعال کنید:

cd path/to/project
pipenv shell



سرور Flask را اجرا کنید:

python -u "C:\xampp\htdocs\New folder\app.py"



راه‌اندازی سرور PHP:

مرورگر را باز کرده و به آدرس http://localhost:8000/upload.php بروید.
